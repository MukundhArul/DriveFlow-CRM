## ⚠️ TROUBLESHOOTING:

### **Browser shows blank screen?**
- Check browser console (F12) for errors
- Make sure you're on `http://localhost:5173/` exactly
- Try clearing cache (Ctrl + Shift + R)

### **Changes not showing up?**
- Vite hot-reloads automatically
- If stuck, stop server (Ctrl+C) and run `npm run dev` again

### **Port 5173 already in use?**
- Stop other dev servers
- Or Vite will auto-pick another port (check terminal output)
